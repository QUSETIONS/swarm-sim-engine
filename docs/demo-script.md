# Swarm Simulation MVP: Demo Script

1. **Introduction**: "Welcome. This is a multi-agent swarm simulation MVP built with a graph-first architecture."
2. **Ingestion**: "We start by uploading source documents, which generate domain-specific ontologies via LLM."
3. **World Graph**: "The system builds a knowledge graph from those documents to anchor our agents."
4. **Agents**: "We spawn autonomous conversational agents directly from graph entities."
5. **Simulation**: "We run a deterministic simulation to see how discussion evolves."
6. **Report & Interview**: "Finally, we generate an interpretable report and allow the user to interview any agent post-run."
